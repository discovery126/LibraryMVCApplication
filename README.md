# Система управления библиотекой

Один из самых ранних учебных проектов на Spring MVC и Java.

## Технологии

- Java 17
- Spring MVC
- Hibernate / JPA
- PostgreSQL
- Apache Tomcat
- Thymeleaf
- Jakarta Validation

## Возможности
- CRUD для книг и пользователей
- Назначение и возврат книг пользователям
- Валидация книг и пользователей
